# Aurora Ai Real Estate

> AI Real Estate Analyzer - Aurora Market APK/AAB | auroramarket.org



## Features

- 🤖 AI-powered core features via OpenRouter
- 🔐 Authentication with Supabase
- 💳 Payments with RevenueCat
- 📱 React Native / Expo mobile app
- 🎨 Modern UI with Tailwind CSS

## Quick Start

```bash
git clone https://github.com/21leahcimhtiek-oss/aurora-ai-real-estate
cd aurora-ai-real-estate
cp .env.example .env.local
npm install
npm run dev
```

## Environment Variables

See `.env.example` for all required environment variables.

## Tech Stack

- React Native / Expo
- Supabase (auth + database)
- OpenRouter (AI)
- RevenueCat (subscriptions)
- TypeScript
